# CodePortal Sign-Up Page 💻

A modern and responsive developer sign-up page built using HTML and CSS.

## 🚀 Features

* Clean, modern UI design
* Two-panel layout (branding + form)
* Responsive structure
* Custom typography (Inter font)
* Developer-focused branding

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts

## 📸 Preview

A sleek sign-up experience designed for a fictional platform called **CodePortal**.

## 📂 Project Structure

```
codeportal-signup-page/
│── index.html
│── styles.css
│── images/
```

## ▶️ How to Run

1. Clone the repository:

   ```
   git clone https://github.com/your-username/codeportal-signup-page.git
   ```
2. Open `index.html` in your browser.
