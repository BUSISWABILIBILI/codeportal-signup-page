# CodePortal Sign-Up Page 💻

A modern, responsive developer sign-up page built using HTML and CSS. This project focuses on clean UI design and a user-friendly onboarding experience for a fictional platform called **CodePortal**.

## 🚀 Features

- Clean and modern UI design
- Two-panel layout (branding + form)
- Fully responsive structure
- Custom typography using Inter font
- Developer-focused branding

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts (Inter)

## 📸 Preview

A sleek and minimal sign-up experience designed to simulate a real-world developer platform.
